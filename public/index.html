Dashboard Components

  • Header Section: Gradient background with user greeting and notifications icon
  • Child Avatars: Horizontal scrollable row of child profiles with an option to add more
  • Consultation Widget: Displays 1-2 recent consultations with status indicators (Active, Waiting Response, Completed)
  • Blog & FAQ Section: Horizontally scrollable cards featuring age-specific content based on children's profiles
  • New Consultation Button: Prominent CTA for starting a new consultation
  • Empty State: Informative guidance for first-time users with no consultations yet
  • Bottom Navigation: Five tabs for Beranda, Anak, Ahli, Riwayat, and Profil sections

Design Principles Applied

  • Elegant Minimalism: Clean layout with ample white space and focused elements
  • Refreshing Gradient Colors: Soft gradients using the brand palette for warmth and professionalism
  • Clear Information Hierarchy: Sectioned content with clear headers and supportive visuals
  • Refined Rounded Corners: Consistent card and button styling with 12px border radius
  • Functional Design: Focused on core functionalities with intuitive navigation patterns
  • Immersive Experience: Smooth transitions between content sections with clear visual cues
  • Comfortable Proportions: Balanced spacing and sizing of UI elements for easy interaction
